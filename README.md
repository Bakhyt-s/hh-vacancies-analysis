# HH Vacancies Analysis

Анализ вакансий с HeadHunter (49 197 вакансий, 23 501 работодатель) с фокусом на рынок труда для Data Scientist-специалистов.

## О проекте

Учебный проект в рамках курса Data Science (SkillFactory). Работа с базой данных PostgreSQL через `psycopg2` и `pandas`: от знакомства со схемой до предметного анализа требований к DS-вакансиям (ключевые навыки, зарплатные вилки по опыту, junior/non-junior сегментация).

## Что внутри

- Знакомство с данными и предварительный анализ (регионы, зарплатные вилки, тип занятости)
- Анализ работодателей (топ по количеству вакансий, географии, сферам деятельности)
- Предметный анализ вакансий Data Scientist (фильтрация по ключевым словам в названии, junior-сегмент, топ ключевых навыков, зарплата по уровню опыта)
- Выводы по каждому блоку + общий вывод по рынку труда

## Стек

`PostgreSQL` · `psycopg2` · `pandas` · Jupyter Notebook

## Как запустить

1. Установить зависимости: `pip install psycopg2 pandas`
2. Заполнить параметры подключения к своей базе PostgreSQL (`DBNAME`, `USER`, `PASSWORD`, `HOST`, `PORT`) в первой ячейке — в текущей версии там плейсхолдеры
3. Выполнить ячейки последовательно сверху вниз

## Примечание

В блоке 6 (анализ ключевых навыков) три задания дали воспроизводимый, логически обоснованный результат, отличающийся от эталонного ответа учебной платформы — вероятная причина в неоднородности исходных данных поля `key_skills` (разные разделители, объединённые технологии вида `MySQL`/`SQL`). Подробности — в соответствующих markdown-комментариях ноутбука.
